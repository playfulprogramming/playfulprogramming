## Why This Matters

According to the [Green Software Foundation](https://greensoftware.foundation/), software contributes significantly to global carbon emissions ‚Äî from data transfer and rendering to power usage on user devices.

Small optimizations in your frontend code ‚Äî like compressing images or enabling dark mode ‚Äî can **actually reduce electricity consumption** at scale.

Green Software Indicator is a gentle reminder of how our decisions as developers have real-world consequences.
